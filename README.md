# K-GPT 🤖

K-GPT is a modern AI-powered chat application built with **Next.js 16**, **React 19**, **Prisma**, **PostgreSQL**, **TanStack Query**, **Clerk Authentication**, and the **Vercel AI SDK**.

## 🚀 Live Demo

**🌐 Live Application:** https://https://kgpt.netlify.app/


## ✨ Features

* 🔐 User authentication with Clerk
* 💬 ChatGPT-like conversation interface
* 📝 Persistent chat history
* 📂 Multiple conversations
* 🌿 Branch conversations
* 🤖 AI-powered responses
* 🌙 Dark & Light mode
* ⚡ Fast and responsive UI
